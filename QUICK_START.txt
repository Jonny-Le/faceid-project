📘 QUICK_START.txt – PyTorch RTX 5090 Environment Guide  
(English + Vietnamese bilingual)

───────────────────────────────────────────────────────────────
🚀 1️⃣ Activate Environment / Kích hoạt môi trường
───────────────────────────────────────────────────────────────
source ~/venv-pytorch/bin/activate
cd ~/myprojects/faceid-project

👉 Loads your Python virtual environment and enters your project folder.  
👉 Bật môi trường ảo và chuyển đến thư mục dự án.

───────────────────────────────────────────────────────────────
⚙️ 2️⃣ Verify GPU & Driver (optional)
───────────────────────────────────────────────────────────────
python src/verify_env.py

✅ Expect output like:
Environment is READY  
GEMM FP16 ~220 TFLOP/s  
torch.compile: OK

✅ Kết quả mong đợi: GPU RTX 5090 hoạt động tốt, không lỗi CUDA/cuDNN.

───────────────────────────────────────────────────────────────
📁 3️⃣ Project Layout / Cấu trúc dự án
───────────────────────────────────────────────────────────────
~/myprojects/faceid-project/
├── src/
│   ├── verify_env.py       ← GPU environment verification
│   ├── train.py            ← to be created for model training
│   ├── model.py            ← neural network definition
│   └── dataset.py          ← dataset loading and augmentation
├── README.txt              ← environment setup notes
└── env-freeze/
    ├── requirements-lock.txt
    └── wheelhouse/         ← offline package storage

───────────────────────────────────────────────────────────────
📦 4️⃣ Offline Install / Cài đặt offline
───────────────────────────────────────────────────────────────
python -m pip install --no-index \
  --find-links ~/env-freeze/wheelhouse \
  -r ~/env-freeze/requirements-lock.txt

➡️ Use this to reinstall environment offline.  
➡️ Dùng khi muốn cài lại môi trường mà không cần Internet.

───────────────────────────────────────────────────────────────
🧠 5️⃣ Start Development / Bắt đầu phát triển
───────────────────────────────────────────────────────────────
- Code trong thư mục `src/`
- Sử dụng `torch.compile`, `channels_last`, và TF32 để đạt tốc độ tối đa
- Bạn có thể tạo `train.py`, `inference.py`, hoặc thử mô hình CNN đầu tiên.

───────────────────────────────────────────────────────────────
🧩 6️⃣ Useful Commands / Lệnh hữu ích
───────────────────────────────────────────────────────────────
nvidia-smi                      ← Kiểm tra GPU  
nvidia-smi --loop=5             ← Theo dõi GPU mỗi 5 giây  
nvidia-smi | grep "Driver Version"  ← Kiểm tra phiên bản driver
